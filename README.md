## covid19.swift

Quick iOS app to show some COVID-19 / health news and tweets :zap:

![](Images/covid19.swift.png)

## Requirements

iOS 13 for [SF Symbols](https://developer.apple.com/design/human-interface-guidelines/sf-symbols/overview/).
